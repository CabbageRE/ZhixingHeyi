## 项目简介

「知行合一——知识开源计划」是一个收集、整理并开源部分学校寒假作业/练习/试卷参考答案的个人学习项目。

项目名称取自王阳明先生「知行合一」思想，希望学习者在获取知识的同时，也能真正践行所学，而不是停留在「知道」的层面。此名称含义仅此而已。

目前主要包含（但不限于）：

- 部分学校寒假作业参考答案
- 部分学科练习/试卷解析

## 使用须知

1. **仅限自查自纠**  
   建议先独立完成后再对照参考，而不是直接抄答案。

2. **请自行判断正确性**  
   本仓库答案并非官方标准答案，可能存在错误或多种解法，仅代表整理者当时的理解。

3. **尊重原题版权**  
   本项目不提供原始题目图片/扫描件，仅提供文字解析或答案，尽量避免直接侵犯试卷版权。

4. **欢迎指正错误**  
   如果发现答案有误，欢迎提交 Issue 或 Pull Request 进行更正。
